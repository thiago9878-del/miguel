# Portafolio Profesional - Thiago

Este proyecto es un portafolio web personal e interactivo desarrollado para mostrar habilidades técnicas, experiencia laboral e intereses personales. La aplicación está construida utilizando un enfoque moderno de desarrollo web con Python.

## 🚀 Tecnologías Utilizadas

El proyecto emplea las siguientes herramientas y lenguajes:

*   **Backend:** [Python](https://www.python.org/) con el micro-framework [Flask](https://flask.palletsprojects.com/).
*   **Frontend:** HTML5 semántico y CSS3 con variables personalizadas.
*   **Interactividad:** JavaScript para efectos en las tarjetas de habilidades y navegación suave.
*   **Iconografía:** [Font Awesome](https://fontawesome.com/) para iconos sociales y de habilidades.
*   **Tipografía:** [Google Fonts](https://fonts.google.com/) (Montserrat y Roboto).
*   **Recursos Visuales:** Fondos temáticos de alta calidad integrados vía URL.

## ✨ Características Principales

1.  **Diseño Responsive:** Adaptable a dispositivos móviles y computadoras de escritorio.
2.  **Secciones Dinámicas:** 
    *   **Sobre Mí:** Introducción personal y perfil social.
    *   **Habilidades Técnicas:** Tarjetas interactivas que revelan descripciones al hacer clic.
    *   **Intereses:** Visualización de pasatiempos y pasiones personales.
    *   **Experiencia Laboral:** Historial detallado de roles anteriores.
    *   **Propuesta de Valor:** Sección dedicada a resaltar la responsabilidad y versatilidad.
3.  **Barra de Navegación Fija:** Navegación fluida (Smooth Scroll) entre las distintas secciones del sitio.
4.  **Anuncios Simulados:** Bloques laterales con temática informática para un diseño tipo "portal tecnológico".

## 🛠️ Instalación y Ejecución Local

Para ejecutar este proyecto en tu máquina local, sigue estos pasos:

1.  **Clonar el repositorio:**
    ```bash
    git clone https://github.com/thiago9878-del/miguel.git
    cd miguel
    ```

2.  **Instalar Flask:**
    ```bash
    pip install Flask
    ```

3.  **Ejecutar la aplicación:**
    ```bash
    python app.py
    ```

4.  **Ver el sitio:**
    Abre tu navegador y entra a `http://127.0.0.1:5000`
